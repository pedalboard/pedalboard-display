The Display PCB for the Open Pedalboard system features 8 RGB LED rings
and 2 OLED display ports for real-time effect parameter visualization.

## Features

- 8 cascaded RGB LED rings (12 individually addressable LEDs each)
- Single serial line addressing (Stemma 3-pin JST-PH)
- 2 additional single LEDs
- 2 ports for 4-wire OLED displays (Stemma 4-pin connectors)
