# Pedalboard Display

LED ring and OLED display PCB for the Open Pedalboard system.

**[Documentation →](https://pedalboard.github.io/pedalboard-display/latest/)**

![Top View](https://pedalboard.github.io/pedalboard-display/latest/3D/pedalboard-display-3D_blender_top.png)

## Features

- 8 cascaded RGB LED rings (12 individually addressable LEDs each)
- Single serial line addressing (Stemma 3-pin JST-PH)
- 2 additional single LEDs
- 2 ports for 4-wire OLED displays (Stemma 4-pin connectors)
